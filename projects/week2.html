Week 2 Project

AI Professional Email & Letter Generator

Overview

The AI Professional Email & Letter Generator is a generative AI web application designed to create high-quality professional communication such as emails, cover letters, motivational letters, and meeting requests.

Purpose

The system helps students, graduates, and professionals generate polished business communication quickly using AI and prompt engineering.

Core Features

  • Email generation
  • Cover letter generation
  • Meeting request generation
  • Motivational letter generation
  • Prompt Library system
  • Copy and download support

Technologies Used

  • React + TypeScript
  • Tailwind CSS
  • Google Gemini AI
  • Prompt Engineering
  • Lovable Cloud
