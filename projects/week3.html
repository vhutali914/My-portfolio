Week 3 Project

SentiCine.AI - Movie Review Sentiment Analyzer

Project Overview

SentiCine.AI is an AI-powered sentiment analysis dashboard designed to analyze movie and TV series reviews. It transforms audience opinions into visual insights using charts, AI summaries, and sentiment classification.

Objectives

  • Analyze audience sentiment
  • Generate AI-powered insights
  • Visualize review trends
  • Allow CSV dataset uploads
  • Create a modern analytics dashboard

Features

  • Live sentiment analyzer
  • Dashboard charts
  • Dataset upload system
  • AI-generated summaries
  • Theme analysis

Technologies Used

  • React 19
  • Tailwind CSS
  • Recharts
  • Google Gemini AI
  • TypeScript
