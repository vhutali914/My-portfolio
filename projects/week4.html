Week 4 Project

EduMind AI - Smart Study Platform

Project Overview

EduMind AI is a modern AI-powered study platform designed for Matric and university students. The platform combines AI tutoring, note summarization, quiz generation, wellness support, and study planning into one calm, student-focused environment.

Main Features

  • AI Study Assistant
  • AI Quiz Generator
  • Notes Summarizer
  • Wellness Companion
  • Smart Planner
  • Analytics Dashboard

Technologies Used

  • React 19
  • TanStack Start
  • Tailwind CSS
  • Lovable AI Gateway
  • Google Gemini
  • Cloudflare Workers
